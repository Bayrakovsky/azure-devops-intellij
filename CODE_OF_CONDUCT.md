# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version:

- Be open, welcoming, and respectful.
- Harassment, insults, and personal attacks are not tolerated.
- Respect differing viewpoints and experience levels; disagree with ideas, not people.
- Assume good faith and give constructive feedback.

## Scope

Applies to all project spaces — issues, pull requests, discussions, and commit messages.

## Reporting

Report unacceptable behavior to **stanislav.job@gmail.com** with the subject
`[azure-devops-intellij CoC]`. Reports are handled confidentially.
